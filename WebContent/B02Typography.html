Highlight Text

Use the mark element to 형광펜효과 text.

Abbreviations

The abbr element is used to mark up an abbreviation or acronym:

The WHO was founded in 1948.

Description Lists - dl/dt/dd 태그의 효과

The dl element indicates a description list:

타이틀
- 설명구문이 삽입된다
Milk
- white cold drink

Keyboard Inputs

문자열을 키보드와 같이 표현한다! 단축키 표현에 유용하다!

Use ctrl + p to open the Print dialog box.

pre태그 적용하기

pre태그는 Text를 편집한 그대로 브라우저에 출력된다. 

            공개된 영상은 시험운행 기간 중 정지궤도 환경위성이 관측한
            아시아 전역의 미세먼지(PM), 이산화질소(NO2), 아황산가스(SO2),
            오존(O3) 등의                          대기오염물질 자료다. 
            “영상 자료를 통해 미세먼지 농도와 관련된 에어로졸 광학두께(AOD),
            이산화질소(NO2), 
            아황산가스(SO2)와 오존(O3)의
            시간대별 발생, 이동 및 분포 현황을 확인할 수 있다”            고 환경부는 설명했다.
